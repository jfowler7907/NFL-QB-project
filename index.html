Welcome to Our NFL QB App!


What is the purpose of this app?

The purpose of this app is to utilize several front-end and back-end technologies to extract NFL quaterback
draft information as well as NFL quaterback game stats, transform the data into clean, easy to read dictionaries,
and load the data into a sqlite database as well as JSON readable API.

What technologies were used?

The following front-end languages were used:
  • HTML5
  • CSS
  • Bootstrap
The following back-end languages/technologies were used:
  • Flask
  • Sqlite3
  • JSON
  • SQLalchemy
