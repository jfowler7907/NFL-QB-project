Welcome to Our NFL QB App!


What is the purpose of this app?

The purpose of this project is to determine if draft round or pick number are an indicator of a quartback's future success in the NFL.
The quarterback's key stats will be compared to league averages as well as the best quarterbacks in the league as decided by average QBR
and visualized in an easy to use, easy to read, interactive dashboard. You will be able to seach by quarterback name or by draft year.

What technologies were used?

The following front-end languages were used:
  • HTML5
  • CSS
  • JavaScript
The following back-end platforms were used:
  • Flask
  • Sqlite3
  • Heroku
